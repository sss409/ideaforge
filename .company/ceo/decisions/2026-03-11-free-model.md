# Phase 1 を全プラン無料に変更

## 意思決定

発案者向けサービスを Phase 1 では完全無料にする。
Stripe Payment Links は不要。Typeform でアイデア投稿のみ。

## 理由

- まず「精錬レポートに価値を感じるか」の検証が最優先
- 有料にすると投稿のハードルが上がり、サンプル数が集まらない
- 無料で質の高いレポートを提供 → 口コミ・リピート → PMF を確認してから課金

## Phase 1 検証ゲートの変更

| 旧（有料） | 新（無料） |
|-----------|-----------|
| 3-5件の実購入 | 30件以上のアイデア投稿 |
| — | NPS 8+ が 50% 以上 |
| — | リピート率 30% 以上（2回目の投稿） |
| 企業 10社の関心 | 企業 10社の関心（変更なし） |

## 収益化タイミング

- Phase 1: 無料（検証期間）
- Phase 2: フリーミアム導入（基本無料 + プレミアム機能）
- Phase 3: 企業向けマーケットプレイス（本来の収益源）

## 影響

- Stripe 不要 → セットアップ工数削減
- LP から価格セクション変更（全プラン無料）
- 検証ゲートを投稿数 + NPS + リピート率に変更
