# 発案者を永久無料に変更

## 意思決定

発案者（利用者）は全フェーズを通じて永久無料。収益は企業側のみから得る二面マーケットプレイスモデルに転換。

## 旧モデルとの違い

| 項目 | 旧（フリーミアム） | 新（利用者永久無料） |
|------|-------------------|---------------------|
| 発案者 Phase 1 | 無料 | 無料 |
| 発案者 Phase 2 | フリーミアム（課金開始） | **無料（維持）** |
| 発案者 Phase 3 | 有料プラン | **無料（維持）** |
| 収益源 | 発案者 + 企業 | **企業のみ** |
| 発案者の対価 | 精錬レポート + 有料機能 | 精錬レポート + **レベニューシェア** |

## 理由

- 供給側（発案者）の障壁をゼロに保ち、アイデア量を最大化する
- 予算がある側（企業）から収益化する方が自然
- LinkedIn/GitHub/Pixta と同じ実証済みの二面マーケットモデル
- 精錬コスト $0.08-0.18/件は十分に低く、企業課金で回収可能

## 影響

- Phase 2 の「フリーミアム導入」→「企業課金開始」に変更
- Stripe Connect の用途: 発案者課金 → 企業課金 + 発案者レベニューシェア精算
- LP のメッセージ: 「完全無料」を永続的な価値提案として強調可能
- 設計原則に「発案者永久無料」を第1項として追加
