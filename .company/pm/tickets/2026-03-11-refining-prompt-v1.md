# 精錬プロンプト v1 を作成する

## 内容
AI精錬エンジンの5ステップパイプラインのプロンプトを設計・作成する。
Claude Web UI でテスト可能な形式で作成し、コードは書かない。

### 5ステップパイプライン
1. アイデア入力の構造化（自然言語 → 構造化データ）
2. 市場分析（ターゲット、競合、市場規模の推定）
3. 実現可能性評価（技術難易度、必要リソース、タイムライン）
4. 収益モデル提案（価格戦略、マネタイズ方法）
5. レポート生成（投資家向け / 実行者向けの2フォーマット）

## 完了条件
